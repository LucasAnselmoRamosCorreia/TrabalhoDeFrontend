The Flowers Garden - Floricultura Online
Sobre o Projeto
The Flowers Garden é um site moderno e responsivo para uma floricultura virtual, desenvolvido com HTML, CSS e JavaScript. O projeto oferece uma experiência completa de e-commerce especializado em flores e arranjos florais, permitindo aos clientes explorar produtos, adicionar itens ao carrinho e finalizar compras de forma intuitiva.

Funcionalidades Principais
Sistema de Carrinho de Compras
Adicionar/Remover Produtos: Interface intuitiva para gerenciar itens no carrinho

Controle de Quantidades: Aumentar ou diminuir quantidades dos produtos

Cálculo Automático: Subtotal, frete e total calculados em tempo real

Sidebar Interativa: Carrinho deslizante com overlay escuro

Notificações: Feedback visual ao adicionar itens
